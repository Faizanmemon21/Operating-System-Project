# 💻 Operating System Simulator

A Python-based Operating System Simulator developed as a university project. This application demonstrates fundamental operating system concepts by simulating CPU scheduling, memory management, file system operations, and security management using a modular architecture.

---

# Features

- 🖥️ CPU/Disk Scheduling Simulation
- 🧠 Memory Management
- 📂 File System Management
- 🔒 Security Management
- 📊 System Log Generation
- 🧩 Modular Python Architecture
- 📚 Educational OS Simulation

---

# Project Structure

```text
CCP_Source_Code_and_Logs/

├── main_simulator.py      # Main application entry point
├── disk_scheduler.py      # Disk scheduling algorithms
├── memory_manager.py      # Memory management module
├── file_system.py         # File system simulation
├── security_manager.py    # Security management

```

# How to Run

## Requirements

- Python 3.8 or later

---

## Installation

Clone the repository:

```bash
git clone https://github.com/faizanmemon4592-arch/Operating-System-Simulator.git
```

Move into the project directory:

```bash
cd Operating-System-Simulator
```

---

## Run the Project

Navigate to the project folder and execute:

```bash
python main_simulator.py
```

or

```bash
python3 main_simulator.py
```

The simulator will start, allowing you to explore the implemented operating system modules.

---

# Modules

### Disk Scheduler
Implements disk scheduling algorithms for processing disk access requests.

### Memory Manager
Simulates memory allocation and management techniques.

### File System
Provides basic file system operations and management.

### Security Manager
Handles simple security-related operations within the simulator.

### Logs
Stores execution logs generated during simulations.
